What do we do?

What's a web application?

Software that runs within your browser like a website.

Why build and use web applications?

There are many benefits in building and using web applications as opposed to desktop applications:

  • Quicker and cheaper to develop than desktop applications

  • Works on all operating systems with a web browser

  • Mobile ready; no need to develop applications for Android and iOS

  • No need to install new software on your computers

  • Access it from anywhere in the world. When your application is hosted in the cloud, you can use it anywhere you have internet access.
